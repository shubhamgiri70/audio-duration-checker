# Audio Duration Checker

A Python tool that analyzes audio datasets and reports duration statistics.

## Features

- Detects duration of WAV and MP3 files
- Calculates dataset statistics
- Reports:
  - Total files
  - Total duration
  - Average clip length
  - Shortest clip
  - Longest clip

## Requirements

- Python 3
- librosa
- numpy

## Install

```bash
pip install librosa numpy
```
